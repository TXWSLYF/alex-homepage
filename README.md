# alex-homepage

个人主页（Portfolio），基于 [Next.js](https://nextjs.org)（App Router）与 [Tailwind CSS](https://tailwindcss.com) 搭建。

## 本地开发

本项目使用 [pnpm](https://pnpm.io) 管理依赖：

```bash
pnpm install
pnpm dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000)。

其他常用命令：

```bash
pnpm build   # 生产构建
pnpm start   # 启动生产服务
pnpm lint    # ESLint
```
